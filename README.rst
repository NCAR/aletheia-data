==============
aletheia-data
==============



Utility package for accessing test/sample data hosted on ftp.cgd.ucar.edu/archive/aletheia-data.


Installation
------------

aletheia-data can be installed from PyPI with pip:

.. code-block:: bash

    pip install aletheia-data
